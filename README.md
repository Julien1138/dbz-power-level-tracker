# DBZ Power Level Tracker

A Dragon Ball Z-themed Pebble watchface that tracks your daily step count as a power level. Reach the configured step threshold and watch your character transform into a Super Saiyan.

## Features

- Step count displayed as a power level in bright green (black on B&W devices)
- Choose between **Goku** and **Vegeta** as your character
- Character transforms to Super Saiyan at the configured step threshold, with a multi-phase animation and vibration sequence
- Goku plays a stretching animation on wrist shake; Vegeta throws punches
- Configurable Super Saiyan step threshold (default: 7,000 steps)
- Clock display with system time format (12h / 24h)
- Date display
- Battery level indicator with color-coded bar (green / yellow / red)
- Vibration alert when Bluetooth disconnects (configurable)
- Bardock expressions react to character state and animations

## Configuration

Open the Pebble app and tap the settings icon on the watchface to configure:

- **Character** — Goku or Vegeta
- **Super Saiyan threshold** — daily steps required to trigger the transformation
- **Vibrate on disconnect** — toggle vibration when Bluetooth disconnects
